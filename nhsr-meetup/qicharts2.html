To look for signals

The magic is one line of code

qic(startMonth,number,
    data     = crimeaLong,
    chart    = 'run',
    decimals = 0, 

    x.angle  = 90,
    title    = 'Run chart deaths in Crimea War',
    ylab     = 'Number of deaths',
    xlab     = 'Month'
)

With the facet line

qic(startMonth,number,
    data     = crimeaLong,
    chart    = 'run',
    decimals = 0, 
    facets   = ~death_cause,
    x.angle  = 90,
    title    = 'Run chart deaths in Crimea War',
    ylab     = 'Number of deaths',
    xlab     = 'Month'
)
